AUDIO FILE INSTRUCTIONS
========================

Place your Eid nasheed or background music here as:
  nasheed.mp3

Recommended: A soft, instrumental Eid nasheed or oud/nay ambient track.

The AudioPlayer component will automatically detect and play it.
Volume is set to 45% by default (adjustable in components/AudioPlayer.tsx).

If no audio file is placed here, the button will still display but
nothing will play when clicked.
